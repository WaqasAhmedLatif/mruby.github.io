Open up mirb (which stands for Interactive mruby).

• If you’re using macOS open up Terminal and type mirb, then hit enter.
• If you’re using Windows, open up Command Prompt and type mirb, then hit enter.
• If you’re using Linux, open up a shell and type mirb and hit enter.

How to get mruby and build mruby
For windows users
For Mac users
you can download mruby from github
$ git clone https://github.com/mruby/mruby.git
Then build mruby
$ cd mruby
$ rake
After building mruby , you need to set the path for mruby.
From system properties -> choose Environment variables
Edit Path and add mruby installed folder’s bin (Eg: c:\mruby\bin)
export your mruby/bin folder
